# strategic-sales
# 📊 Strategic Sales & Revenue Intelligence Dashboard

An advanced interactive business intelligence dashboard built using Python, Streamlit, Pandas, and Plotly.

This project helps analyze sales performance, revenue trends, regional performance, top-selling products, and business profitability using interactive visualizations and KPI tracking.

---

# 🚀 Features

## ✅ Interactive Dashboard

* Real-time KPI monitoring
* Dynamic filtering system
* Responsive layout
* Interactive business analytics

## ✅ KPI Metrics

* Total Sales
* Gross Profit
* Profit Margin
* Average Daily Sales
* Total Transactions

## ✅ Advanced Visualizations

* Monthly Revenue Trend Analysis
* Pareto Analysis (80/20 Rule)
* Regional Revenue Distribution
* Top Product Performance Table
* What-If Revenue Simulation

## ✅ File Upload Support

Users can upload:

* CSV files
* Excel files (.xlsx)

## ✅ Smart Sample Data Generator

If no dataset is uploaded, the application automatically generates sample business data for demo purposes.

# ⚙️ Installation Guide

## Step 1 — Clone Repository

```bash
git clone https://github.com/your-username/SalesDashboard.git
```

## Step 2 — Open Project Folder

```bash
cd SalesDashboard
```

## Step 3 — Create Virtual Environment

```bash
py -3.12 -m venv venv
```

## Step 4 — Activate Virtual Environment

### Windows PowerShell

```bash
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```

```bash
.\venv\Scripts\activate
```

---

## Step 5 — Install Dependencies

```bash
python -m pip install -r requirements.txt
```

OR

```bash
python -m pip install streamlit pandas plotly scikit-learn numpy openpyxl
```

---

# ▶️ Running the Application

Run the Streamlit dashboard using:

```bash
python -m streamlit run app.py
```

After running successfully, open:

```bash
http://localhost:8501
```
